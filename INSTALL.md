# دليل التثبيت السريع

## المتطلبات الأساسية
- متصفح حديث (Chrome, Firefox, Safari, Edge)
- اتصال إنترنت مستقر
- حساب Firebase (مجاني)

## الخطوات السريعة للتثبيت

### 1. إعداد Firebase (5 دقائق)
1. انتقل إلى [Firebase Console](https://console.firebase.google.com/)
2. أنشئ مشروع جديد
3. أضف تطبيق ويب جديد
4. انسخ معلومات التكوين
5. عدّل ملف `js/firebase-config.js`

### 2. تكوين قاعدة البيانات
1. في Firebase Console، انتقل إلى Realtime Database
2. أنشئ قاعدة البيانات
3. أنشئ قواعد الأمان (انظر README.md)

### 3. الاستضافة

#### خيار أ: GitHub Pages (أسرع)
1. أنشئ مستودع جديد على GitHub
2. رفع جميع الملفات
3. فعل GitHub Pages من الإعدادات
4. استخدم الفرع main

#### خيار ب: Firebase Hosting
1. ثبت Firebase CLI: `npm install -g firebase-tools`
2. `firebase login`
3. `firebase deploy`

### 4. الإعداد الأولي
1. افتح صفحة الإدارة
2. أضف بيانات مركزك
3. أنشئ العيادات
4. اختبر النظام

## اختبار سريع
1. افتح صفحة العرض
2. افتح صفحة التحكم في نافذة أخرى
3. سجل الدخول لعيادة
4. اضغط "العميل التالي"
5. تحقق من النداء على صفحة العرض

## الأخطاء الشائعة
- **Firebase غير متصل**: تأكد من معلومات التكوين
- **لا يعمل الصوت**: تحقق من إعدادات المتصفح
- **كلمة مرور خاطئة**: تأكد من إعدادات العيادة

## الدعم
- تحقق من ملف README.md للتفاصيل الكاملة
- تأكد من اتصال الإنترنت
- استخدم متصفحاً حديثاً
